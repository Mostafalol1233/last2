التفاضل والتكامل
قوانين المشتقات الأساسية
  • $\frac{d}{dx}(c) = 0$ (مشتقة الثابت)
  • $\frac{d}{dx}(x^n) = nx^{n-1}$ (قاعدة الأس)
  • $\frac{d}{dx}(e^x) = e^x$ (مشتقة الدالة الأسية)
  • $\frac{d}{dx}(\ln x) = \frac{1}{x}$ (مشتقة اللوغاريتم الطبيعي)
  • $\frac{d}{dx}(\sin x) = \cos x$ (مشتقة جيب الزاوية)
  • $\frac{d}{dx}(\cos x) = -\sin x$ (مشتقة جيب تمام الزاوية)
  • $\frac{d}{dx}(\tan x) = \sec^2 x$ (مشتقة ظل الزاوية)
قوانين التكامل الأساسية
  • $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (حيث $n \neq -1$)
  • $\int \frac{1}{x} dx = \ln|x| + C$
  • $\int e^x dx = e^x + C$
  • $\int \sin x dx = -\cos x + C$
  • $\int \cos x dx = \sin x + C$
  • $\int \sec^2 x dx = \tan x + C$
  • $\int \frac{1}{1+x^2} dx = \tan^{-1} x + C$
قواعد التفاضل المركبة
  • قاعدة الجمع: $\frac{d}{dx}(f(x) + g(x)) = \frac{d}{dx}f(x) + \frac{d}{dx}g(x)$
  • قاعدة الضرب: $\frac{d}{dx}(f(x) \cdot g(x)) = f'(x)g(x) + f(x)g'(x)$
  • قاعدة القسمة: $\frac{d}{dx}(\frac{f(x)}{g(x)}) = \frac{f'(x)g(x) - f(x)g'(x)}{g(x)^2}$
  • قاعدة السلسلة: $\frac{d}{dx}(f(g(x))) = f'(g(x)) \cdot g'(x)$
تطبيقات التفاضل والتكامل
  • القيمة العظمى: إذا كان $f'(c) = 0$ و $f''(c) < 0$، فإن $f(c)$ قيمة عظمى
  • القيمة الصغرى: إذا كان $f'(c) = 0$ و $f''(c) > 0$، فإن $f(c)$ قيمة صغرى
  • نقطة الانعطاف: إذا كان $f''(c) = 0$ و $f'''(c) \neq 0$، فإن $(c, f(c))$ نقطة انعطاف
  • المساحة تحت المنحنى من $a$ إلى $b$: $\int_a^b f(x)dx$
  • طول المنحنى: $L = \int_a^b \sqrt{1 + [f'(x)]^2} dx$
  • حجم دوران حول محور $x$: $V = \pi \int_a^b [f(x)]^2 dx$
حساب المثلثات
المتطابقات الأساسية
  • $\sin^2 \theta + \cos^2 \theta = 1$
  • $\tan \theta = \frac{\sin \theta}{\cos \theta}$
  • $\cot \theta = \frac{\cos \theta}{\sin \theta} = \frac{1}{\tan \theta}$
  • $\sec \theta = \frac{1}{\cos \theta}$
  • $\csc \theta = \frac{1}{\sin \theta}$
  • $1 + \tan^2 \theta = \sec^2 \theta$
  • $1 + \cot^2 \theta = \csc^2 \theta$
صيغ جمع الزوايا
  • $\sin(\alpha \pm \beta) = \sin \alpha \cos \beta \pm \cos \alpha \sin \beta$
  • $\cos(\alpha \pm \beta) = \cos \alpha \cos \beta \mp \sin \alpha \sin \beta$
  • $\tan(\alpha \pm \beta) = \frac{\tan \alpha \pm \tan \beta}{1 \mp \tan \alpha \tan \beta}$
صيغ الزاوية المضاعفة
  • $\sin 2\theta = 2\sin \theta \cos \theta$
  • $\cos 2\theta = \cos^2 \theta - \sin^2 \theta = 2\cos^2 \theta - 1 = 1 - 2\sin^2 \theta$
  • $\tan 2\theta = \frac{2\tan \theta}{1 - \tan^2 \theta}$
قوانين المثلثات
  • قانون الجيوب: $\frac{\sin A}{a} = \frac{\sin B}{b} = \frac{\sin C}{c}$
  • قانون جيوب التمام: $c^2 = a^2 + b^2 - 2ab\cos C$
  • مساحة المثلث: $\text{المساحة} = \frac{1}{2}ab\sin C$
صيغ نصف الزاوية
  • $\sin \frac{\theta}{2} = \pm\sqrt{\frac{1-\cos \theta}{2}}$
  • $\cos \frac{\theta}{2} = \pm\sqrt{\frac{1+\cos \theta}{2}}$
  • $\tan \frac{\theta}{2} = \frac{\sin \theta}{1+\cos \theta} = \frac{1-\cos \theta}{\sin \theta}$
صيغ تحويل المنتج إلى مجموع
  • $\sin \alpha \sin \beta = \frac{1}{2}[\cos(\alpha-\beta) - \cos(\alpha+\beta)]$
  • $\cos \alpha \cos \beta = \frac{1}{2}[\cos(\alpha-\beta) + \cos(\alpha+\beta)]$
  • $\sin \alpha \cos \beta = \frac{1}{2}[\sin(\alpha+\beta) + \sin(\alpha-\beta)]$
الميكانيكا
قوانين الحركة الخطية
  • السرعة المتوسطة: $\bar{v} = \frac{\Delta x}{\Delta t}$
  • السرعة اللحظية: $v = \frac{dx}{dt}$
  • التسارع المتوسط: $\bar{a} = \frac{\Delta v}{\Delta t}$
  • التسارع اللحظي: $a = \frac{dv}{dt} = \frac{d^2x}{dt^2}$
  • السرعة عند تسارع ثابت: $v = v_0 + at$
  • الإزاحة عند تسارع ثابت: $x = x_0 + v_0t + \frac{1}{2}at^2$
  • العلاقة بين السرعة والإزاحة: $v^2 = v_0^2 + 2a(x - x_0)$
القوى والحركة
  • قانون نيوتن الثاني: $F = ma$
  • الوزن: $W = mg$
  • قوة الاحتكاك: $F_f = \mu N$ (حيث $\mu$ معامل الاحتكاك)
  • الشغل: $W = F \cdot d \cdot \cos\theta$
  • الطاقة الحركية: $KE = \frac{1}{2}mv^2$
  • طاقة الوضع الجاذبية: $PE = mgh$
  • حفظ الطاقة: $KE_1 + PE_1 = KE_2 + PE_2$
الحركة الدائرية
  • السرعة الزاوية: $\omega = \frac{\Delta \theta}{\Delta t}$
  • السرعة الخطية في الحركة الدائرية: $v = r\omega$
  • التسارع المركزي: $a_c = \frac{v^2}{r} = r\omega^2$
  • القوة المركزية: $F_c = \frac{mv^2}{r} = mr\omega^2$
  • العزم الدوراني: $L = I\omega$ (حيث $I$ عزم القصور الذاتي)
  • العزم: $\tau = r \times F = rF\sin\theta$
ميكانيكا الموائع
  • الكثافة: $\rho = \frac{m}{V}$
  • الضغط: $P = \frac{F}{A}$
  • الضغط الهيدروستاتيكي: $P = P_0 + \rho gh$
  • قوة الطفو: $F_B = \rho_f gV$ (مبدأ أرخميدس)
  • معادلة الاستمرارية: $A_1v_1 = A_2v_2$
  • معادلة برنولي: $P_1 + \frac{1}{2}\rho v_1^2 + \rho gh_1 = P_2 + \frac{1}{2}\rho v_2^2 + \rho gh_2$
الديناميكا
الدفع والزخم
  • الزخم الخطي: $p = mv$
  • الدفع: $J = F \Delta t$
  • نظرية الدفع والزخم: $J = \Delta p$
  • حفظ الزخم: $m_1v_{1i} + m_2v_{2i} = m_1v_{1f} + m_2v_{2f}$
  • معامل الاستعادة في التصادم: $e = \frac{v_{2f} - v_{1f}}{v_{1i} - v_{2i}}$
  • تصادم مرن: $e = 1$
  • تصادم غير مرن تمامًا: $e = 0$
الديناميكا الدورانية
  • عزم القصور الذاتي لجسيم: $I = mr^2$
  • عزم القصور الذاتي لقضيب رفيع: $I = \frac{1}{12}mL^2$
  • عزم القصور الذاتي لقرص: $I = \frac{1}{2}mR^2$
  • عزم القصور الذاتي لكرة مجوفة: $I = \frac{2}{3}mR^2$
  • عزم القصور الذاتي لكرة صلبة: $I = \frac{2}{5}mR^2$
  • الطاقة الحركية الدورانية: $KE_{rot} = \frac{1}{2}I\omega^2$
  • نظرية المحاور المتوازية: $I = I_{cm} + md^2$
الاهتزازات والحركة التوافقية البسيطة
  • معادلة الحركة: $x(t) = A\cos(\omega t + \phi)$
  • التردد الزاوي: $\omega = 2\pi f = \sqrt{\frac{k}{m}}$
  • الزمن الدوري: $T = \frac{2\pi}{\omega} = 2\pi\sqrt{\frac{m}{k}}$
  • التردد: $f = \frac{1}{T} = \frac{\omega}{2\pi} = \frac{1}{2\pi}\sqrt{\frac{k}{m}}$
  • ثابت النابض: $k = \frac{F}{x}$
  • طاقة الحركة التوافقية البسيطة: $E = \frac{1}{2}kA^2$
الديناميكا الكهربائية
  • قانون كولوم: $F = k\frac{q_1q_2}{r^2}$
  • المجال الكهربائي: $E = \frac{F}{q} = k\frac{Q}{r^2}$
  • الجهد الكهربائي: $V = \frac{U}{q} = k\frac{Q}{r}$
  • قانون أوم: $V = IR$
  • القدرة الكهربائية: $P = VI = I^2R = \frac{V^2}{R}$
  • قوة لورنتز: $F = qvB\sin\theta$
  • قانون فرادي: $\varepsilon = -\frac{d\Phi_B}{dt}$
