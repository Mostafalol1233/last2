Basic Algebra Rules
Addition and Subtraction Rules

  • a + b = b + a

    Commutative Property of Addition: The order of addition does not change the result

  • (a + b) + c = a + (b + c)

    Associative Property of Addition: Terms can be grouped in any order

  • a + 0 = a

    Identity Element of Addition: Adding zero does not change the value

  • a + (-a) = 0

    Additive Inverse: The sum of any number and its additive inverse equals zero

Multiplication and Division Rules

  • a × b = b × a

    Commutative Property of Multiplication: The order of multiplication does not change the result

  • (a × b) × c = a × (b × c)

    Associative Property of Multiplication: Factors can be grouped in any order

  • a × 1 = a

    Identity Element of Multiplication: Multiplying by one does not change the value

  • a × (1/a) = 1, where a ≠ 0

    Multiplicative Inverse: The product of any non-zero number and its reciprocal equals one

Distributive Property

  • a(b + c) = ab + ac

    Distributive Property of Multiplication over Addition: Multiplying a sum equals the sum of products

  • a(b - c) = ab - ac

    Distributive Property of Multiplication over Subtraction: Multiplying a difference equals the difference of products

Fraction Rules

  • $\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}$

    Addition of Fractions

  • $\frac{a}{b} - \frac{c}{d} = \frac{ad - bc}{bd}$

    Subtraction of Fractions

  • $\frac{a}{b} \times \frac{c}{d} = \frac{ac}{bd}$

    Multiplication of Fractions

  • $\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} = \frac{ad}{bc}$

    Division of Fractions

Exponents and Roots
Laws of Exponents

  • a^m × a^n = a^(m+n)

    Product of Powers with Same Base: Add the exponents

  • a^m ÷ a^n = a^(m-n)

    Quotient of Powers with Same Base: Subtract the exponents

  • (a^m)^n = a^(m×n)

    Power of a Power: Multiply the exponents

  • (a×b)^n = a^n × b^n

    Power of a Product: Distribute the exponent to each factor

  • (a/b)^n = a^n / b^n

    Power of a Quotient: Distribute the exponent to numerator and denominator

  • a^0 = 1 (a ≠ 0)

    Zero Exponent: Any non-zero number raised to the power of zero equals one

  • a^(-n) = 1/(a^n)

    Negative Exponent: Reciprocal of the positive power

Fractional Exponents and Roots

  • a^(1/n) = $\sqrt[n]{a}$

    Reciprocal Exponent: The nth root

  • a^(m/n) = $(\sqrt[n]{a})^m = \sqrt[n]{a^m}$

    General Fractional Exponent

  • $\sqrt{a \times b} = \sqrt{a} \times \sqrt{b}$

    Root of a Product

  • $\sqrt{a / b} = \sqrt{a} / \sqrt{b}$

    Root of a Quotient

  • $(\sqrt{a})^n = a^{n/2}$

    Power of a Square Root

  • $\sqrt[3]{a \times b} = \sqrt[3]{a} \times \sqrt[3]{b}$

    Cube Root Properties

Polynomials
Square and Cube Formulas

  • (a + b)² = a² + 2ab + b²

    Square of a Binomial (Sum)

  • (a - b)² = a² - 2ab + b²

    Square of a Binomial (Difference)

  • (a + b)(a - b) = a² - b²

    Product of Sum and Difference (Difference of Squares)

  • (a + b)³ = a³ + 3a²b + 3ab² + b³

    Cube of a Sum

  • (a - b)³ = a³ - 3a²b + 3ab² - b³

    Cube of a Difference

Additional Formulas

  • a³ + b³ = (a + b)(a² - ab + b²)

    Sum of Cubes

  • a³ - b³ = (a - b)(a² + ab + b²)

    Difference of Cubes

  • (a + b + c)² = a² + b² + c² + 2ab + 2ac + 2bc

    Square of a Trinomial

  • (a + b)(c + d) = ac + ad + bc + bd

    Product of Two Binomials

  • (a + b)(a² - ab + b²) = a³ + b³

    Alternative Form for Sum of Cubes

Factoring
Basic Factoring Rules

  • a² - b² = (a + b)(a - b)

    Difference of Squares

  • a² + 2ab + b² = (a + b)²

    Perfect Square Trinomial (Sum)

  • a² - 2ab + b² = (a - b)²

    Perfect Square Trinomial (Difference)

  • a³ - b³ = (a - b)(a² + ab + b²)

    Difference of Cubes

  • a³ + b³ = (a + b)(a² - ab + b²)

    Sum of Cubes

Factoring Quadratic Polynomials

  • ax² + bx + c = a(x - r₁)(x - r₂)

    General Form (r₁, r₂ are the roots of the equation)

  • x² + bx + c = (x + p)(x + q), where p + q = b and p × q = c

    Trial and Error Method (sum of coefficients = coefficient of x, product = constant)

  • ax² + bx + c = a(x + b/2a)² + (c - b²/4a)

    Completing the Square

Factoring Polynomials

  • P(x) = a(x - r₁)(x - r₂)...(x - rₙ)

    General Form (r₁, r₂, ... rₙ are the roots)

  • P(r) = 0 ⟺ (x - r) is a factor of P(x)

    Remainder Theorem: If P(r) = 0, then (x - r) is a factor of P(x)

  • If P(x) = (x - r)Q(x) + R, then P(r) = R

    General Remainder Theorem

Rational Root Theorems

  • If P(x) = aₙxⁿ + ... + a₁x + a₀, and r is a root, then r divides a₀

    Constant Term Theorem: Roots of P(x) are among the factors of the constant term a₀

  • If P(x) = aₙxⁿ + ... + a₁x + a₀, and r = p/q in lowest form, then p divides a₀ and q divides aₙ

    Rational Roots Theorem

Equations
Linear Equations

  • ax + b = 0

    x = -b/a

    General Form and Solution

  • a₁x + b₁ = a₂x + b₂

    x = (b₂ - b₁)/(a₁ - a₂)

    Equation with Variable on Both Sides

Quadratic Equations

  • ax² + bx + c = 0

    x = (-b ± √(b² - 4ac)) / 2a

    Standard Form and Quadratic Formula (General Formula)

  • Discriminant (Δ) = b² - 4ac

    Δ > 0: two real roots

    Δ = 0: one repeated real root

    Δ < 0: two complex conjugate roots

    Discriminant and Nature of Roots

  • For ax² + bx + c = 0 with roots r and s:

    Sum of roots: r + s = -b/a

    Product of roots: r × s = c/a

    Relationship Between Roots and Coefficients

Systems of Equations

  • For a system of two linear equations:

    a₁x + b₁y = c₁

    a₂x + b₂y = c₂

    The solution is:

    x = (c₁b₂ - c₂b₁)/(a₁b₂ - a₂b₁)

    y = (a₁c₂ - a₂c₁)/(a₁b₂ - a₂b₁)

    Solving a System of Two Linear Equations (Cramer's Rule)

  • Determinant (D) = a₁b₂ - a₂b₁

    D ≠ 0: unique solution

    D = 0: no solution or infinitely many solutions

    Determinant and Type of Solution

Additional Equations

  • a^x = b

    x = log_a(b)

    Exponential Equations

  • log_a(x) = b

    x = a^b

    Logarithmic Equations

  • |ax + b| = c

    ax + b = c or ax + b = -c

    Absolute Value Equations
