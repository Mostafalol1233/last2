Basic Algebra Rules
Addition and Subtraction Rules

  • a + b = b + a

    Commutative Property of Addition: The order of addition does not change the result

  • (a + b) + c = a + (b + c)

    Associative Property of Addition: Terms can be grouped in any order

  • a + 0 = a

    Identity Element of Addition: Adding zero does not change the value

  • a + (-a) = 0

    Additive Inverse: The sum of any number and its additive inverse equals zero

Multiplication and Division Rules

  • a × b = b × a

    Commutative Property of Multiplication: The order of multiplication does not change the result

  • (a × b) × c = a × (b × c)

    Associative Property of Multiplication: Factors can be grouped in any order

  • a × 1 = a

    Identity Element of Multiplication: Multiplying by one does not change the value

  • a × (1/a) = 1, where a ≠ 0

    Multiplicative Inverse: The product of any non-zero number and its reciprocal equals one

Distributive Property

  • a(b + c) = ab + ac

    Distributive Property of Multiplication over Addition: Multiplying a sum equals the sum of products

  • a(b - c) = ab - ac

    Distributive Property of Multiplication over Subtraction: Multiplying a difference equals the difference of products

Fraction Rules

  • $\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}$

    Addition of Fractions

  • $\frac{a}{b} - \frac{c}{d} = \frac{ad - bc}{bd}$

    Subtraction of Fractions

  • $\frac{a}{b} \times \frac{c}{d} = \frac{ac}{bd}$

    Multiplication of Fractions

  • $\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} = \frac{ad}{bc}$

    Division of Fractions

Exponents and Roots
Laws of Exponents

  • a^m × a^n = a^(m+n)

    Product of Powers with Same Base: Add the exponents

  • a^m ÷ a^n = a^(m-n)

    Quotient of Powers with Same Base: Subtract the exponents

  • (a^m)^n = a^(m×n)

    Power of a Power: Multiply the exponents

  • (a×b)^n = a^n × b^n

    Power of a Product: Distribute the exponent to each factor

  • (a/b)^n = a^n / b^n

    Power of a Quotient: Distribute the exponent to numerator and denominator

  • a^0 = 1 (a ≠ 0)

    Zero Exponent: Any non-zero number raised to the power of zero equals one

  • a^(-n) = 1/(a^n)

    Negative Exponent: Reciprocal of the positive power

Fractional Exponents and Roots

  • a^(1/n) = $\sqrt[n]{a}$

    Reciprocal Exponent: The nth root

  • a^(m/n) = $(\sqrt[n]{a})^m = \sqrt[n]{a^m}$

    General Fractional Exponent

  • $\sqrt{a \times b} = \sqrt{a} \times \sqrt{b}$

    Root of a Product

  • $\sqrt{a / b} = \sqrt{a} / \sqrt{b}$

    Root of a Quotient

  • $(\sqrt{a})^n = a^{n/2}$

    Power of a Square Root

  • $\sqrt[3]{a \times b} = \sqrt[3]{a} \times \sqrt[3]{b}$

    Cube Root Properties

Polynomials
Square and Cube Formulas

  • (a + b)² = a² + 2ab + b²

    Square of a Binomial (Sum)

  • (a - b)² = a² - 2ab + b²

    Square of a Binomial (Difference)

  • (a + b)(a - b) = a² - b²

    Product of Sum and Difference (Difference of Squares)

  • (a + b)³ = a³ + 3a²b + 3ab² + b³

    Cube of a Sum

  • (a - b)³ = a³ - 3a²b + 3ab² - b³

    Cube of a Difference

Additional Formulas

  • a³ + b³ = (a + b)(a² - ab + b²)

    Sum of Cubes

  • a³ - b³ = (a - b)(a² + ab + b²)

    Difference of Cubes

  • (a + b + c)² = a² + b² + c² + 2ab + 2ac + 2bc

    Square of a Trinomial

  • (a + b)(c + d) = ac + ad + bc + bd

    Product of Two Binomials

  • (a + b)(a² - ab + b²) = a³ + b³
